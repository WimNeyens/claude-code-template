Append a new task to `TASKS.md` at the repo root.

Arguments: the task description (everything after `/task-add`).

Steps:
1. If `TASKS.md` does not exist, create it with the standard header (see the template in the repo).
2. Append a new line: `- [ ] <description>`
3. Confirm to the user with the new line.

Do not commit. The user will commit when ready.
