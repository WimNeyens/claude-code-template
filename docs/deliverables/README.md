# Deliverables

Finished artifacts intended for an audience outside the working team: reports, presentations, executive summaries, final documents.

Subfolders:
- `presentations/` — slide decks and talking points

Conventions:
- Name deliverables with a date: `2026-04-06-steerco-update.md`
- State the audience at the top (see `.claude/rules/documentation.md`)
- Link back to supporting material in `docs/analysis/` rather than duplicating content
